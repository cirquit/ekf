Detailed Description

template<size_t N, size_t M>
class kafi::jacobian_function< N, M >

A wrapper function that stores a function and its jacobian.

This is currently working with references of the return values so this class doesn't have any ownership over the stored data

Template arguments:

  • N = state dimensions
  • M = sensor dimensions

For examples, see tests/jacobian_function_tests.cc
